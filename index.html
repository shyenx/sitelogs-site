What it does

Active milliseconds per visit, per URL, per domain, per category. Idle ≥60s automatically pauses tracking, so the number is real focus, not "tab open duration". A GitHub-style grid showing when you actually browse. Switch between calendar years from one dropdown. Continuous reading sessions ≥10 min surfaced as "best work today". Set per-category daily focus budgets with progress bars. Today vs your average Tuesday, this week vs last week, with weekday baselines. New / lapsed domains show what you started and what you dropped. Dedupe by URL, group by domain into Chrome's native colored tab groups. Pinned tabs untouched. Save the current window as a named Session, restore later. Tag any URL/domain. Custom domain or regex rules override the built-in 200+ category dictionary. Filter History by tag.

Privacy by design

  • 🚫 No telemetry. No analytics. No external server.
  • 📦 All visit data lives in your browser's IndexedDB on this device.
  • 🔄 The only outbound call is Google Fonts CSS (will self-host). Once paid licensing launches, license activation will add a second call to our payment provider — nothing else, ever.
  • 📤 One click in Settings exports everything to JSON, or wipes it.
  • 📜 We commit to a clear privacy policy and never adding hidden trackers.
